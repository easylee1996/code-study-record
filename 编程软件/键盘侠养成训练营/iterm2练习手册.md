## 导航

- 分屏功能：
  - 左右：`cmd + d`
  - 上下：`cmd + shift + d`
  - 切换：`cmd + []`左右切换，不够用的话，`cmd + option + 方向键`切换上下左右。
  - 调整分屏窗口的宽度：`ctrl + cmd + 方向键`
- 标签页：
  - 新建：`cmd + t`
  - 关闭：`cmd + w`，关闭标签必须先关闭里面的窗口。
  - 切换：`cmd + 数字`或者 `cmd + 方向键`
  - 移动当前标签页到指定位置：`cmd + shift + 方向键`
- 历史记录：
  - iterm自带：`ctrl + p`、`ctrl + n`或者上下键
